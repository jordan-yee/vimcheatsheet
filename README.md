# Vim
### Files

| **Command or Key** | **Description** |
|:---------------|:------------|
| :edit | Edit a file. |
| :edit $MYVIMRC | Edit your vimrc file. |
| :edit .\dir | Open directory with netrw. |
| :hide edit | Edit a different file while retaining changes in current file. |
